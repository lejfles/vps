## E. Code Modifications (recommended)

### File: buy_iphone page render
- **Why**: 4 H1s + missing Product schema
- **Change 1**: Reduce H1s from 4 → 1, demote 3 to H2
- **Change 2**: Inject Product JSON-LD before `</head>`
- **Schema example**:
```json
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "iPhone 15 Pro",
  "brand": {"@type": "Brand", "name": "Apple"},
  "offers": {
    "@type": "Offer",
    "priceCurrency": "USD",
    "price": "999.00",
    "availability": "https://schema.org/InStock"
  }
}
```
- **SEO impact**: Rich product result eligibility in Google Search; clearer page hierarchy
- **Risk**: P1

### File: macbook_pro page
- **Why**: 145/217 images without alt
- **Change**: Add alt text to all `<img>` tags (decorative: `alt=""`, informational: descriptive alt)
- **SEO impact**: Better image search ranking + WCAG AA compliance + LLM extractability
- **Risk**: P1

### New file: /llms.txt
- **Why**: Apple.com has no llms.txt (404), even though they should set the standard
- **Change**: Create `/llms.txt` with:
```
# Apple

Apple designs and markets consumer electronics, computer software, and online services.

## Key Products
- [iPhone](https://www.apple.com/iphone/): Smartphone lineup
- [MacBook Pro](https://www.apple.com/macbook-pro/): Pro laptop with M3 family chips
- [iPad](https://www.apple.com/ipad/): Tablet lineup
- ... (etc)

## Support
- [Apple Support](https://support.apple.com): Official support portal
- [Contact Apple](https://www.apple.com/contact/): Sales, support, accessibility
```
- **SEO impact**: Improved AI-crawler discovery for ChatGPT / Perplexity / Claude citation
- **Risk**: P2

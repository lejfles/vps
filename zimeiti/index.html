用法

查看话题大全

点击上方任一话题卡片进入对应 HTML。每个 HTML 包含：话题定义、权威资料补齐、已归类的视频脚本（带日期/作者/英文 transcript 节选）、更新日志。

每日自动归类

每天收完 2 条新视频脚本后，跑一次：

python3 classify.py

脚本会自动扫描 ~/zimeiti_tiktok_monitor/outputs/，按关键词匹配归入对应话题，追加到对应 HTML 的"已归类脚本"和"更新日志"两节。已处理过的文件通过 SHA256 + HTML marker 双重去重，不会重复。

新增话题怎么办

  1. 在 topics_html/ 下新建 HTML，模板可参考现有 7 个
  2. HTML 必须含两个固定锚点：<h2>本话题已归类的视频脚本</h2> 和 <h2>更新日志</h2>
  3. 在 classify.py 的 TOPIC_RULES 中加新规则
  4. 重跑 classify.py，新话题会开始被填充
